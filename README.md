Este projeto utiliza o dataset Heart Failure Prediction Dataset para prever a probabilidade de um paciente sofrer uma insuficiência cardíaca com base em 11 variáveis clínicas (como idade, colesterol, pressão arterial, entre outras).

🧪 Algoritmos Utilizados
Para esta análise, comparamos o desempenho de duas famílias distintas de classificadores:

Modelos Baseados em Árvores: (Ex: Decision Tree ou Random Forest). Modelos que segmentam os dados através de regras de decisão hierárquicas.

Modelos Probabilísticos: (Ex: Naive Bayes). Modelos baseados no Teorema de Bayes que assumem independência entre os atributos para calcular a probabilidade de cada classe.
